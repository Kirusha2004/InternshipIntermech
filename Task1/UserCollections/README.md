# Задание 2: Коллекция месяцев

## Описание задачи
Создать коллекцию, хранящую информацию о 12 месяцах: наименование, порядковый номер и количество дней. Реализовать возможность выборки месяцев по номеру и количеству дней.

## Решение
- Создан класс `Month` с свойствами: Number, Name, Days
- Реализована коллекция List<Month> с данными всех месяцев
- Использованы LINQ-запросы для фильтрации:
  - По номеру месяца
  - По количеству дней
  - Комбинированные условия

## Примеры использования
```csharp
// Выборка по номеру
var monthByNumber = list.Where(m => m.Number == 5);

// Выборка по количеству дней
var monthsWith31Days = list.Where(m => m.Days == 31);

// Комбинированная выборка
var combinedResult = list.Where(m => m.Number > 6 && m.Days == 30);
