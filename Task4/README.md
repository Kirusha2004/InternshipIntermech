Задание 3: Коллекция счетов
Описание задачи
Создать коллекции для хранения информации о банковских счетах (номер счета и баланс). Реализовать различные операции с использованием LINQ для фильтрации, сортировки и агрегации данных.

Решение
Использованы три типа коллекций:

Dictionary<int, double> для хранения пар номер-баланс

SortedDictionary<int, double> для автоматической сортировки по ключам

SortedList<int, double> для эффективного доступа по отсортированным ключам

Реализованы LINQ-запросы для:

Фильтрации по балансу и номеру счета

Сортировки по убыванию баланса

Агрегатных операций (сумма, максимум, среднее)

Преобразования валют (конвертация в евро)

Примеры использования
```csharp

// Счета с балансом выше 20000
var highBalanceAccounts = accounts.Where(a => a.Value > 20000);

// Топ счетов по балансу
var topAccounts = accounts.OrderByDescending(a => a.Value);

// Общий баланс
var totalBalance = accounts.Sum(a => a.Value);

// Конвертация в евро
var euros = accounts.ToDictionary(a => a.Key, a => a.Value * 0.85);
```
